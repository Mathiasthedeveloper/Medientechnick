CSS in HTML einbinden

Variante 1: Inline Attribut

Man verändert die Gestaltung mit dem style-Attribut.



Variante 2: Internes Stylesheet

Man verändert die Gestaltung durch CSS-Code innerhalb des style-Tags im Kopfbereich der HTML-Datei.



Variante 3: Externes Stylesheet

Man verändert die Gestaltung durch CSS-Code in einer separaten CSS-Datei.
